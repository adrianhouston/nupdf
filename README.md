## Why NuPDF?

NuPDF is not about overwhelming you with features. It's about creating a calm, focused reading experience that feels natural and comfortable. Whether you're reading for work or leisure, NuPDF helps you stay in the zone.

## Download

You can download the latest debug APKs from the [`extras`](./extras/) folder.  

> [!WARNING]
> These are debug builds for testing and may not be suitable for production.

## Build

```sh
fish bootstrap.fish --update --patch --build
```

## Features

* PDF Dark Mode follows system settings
* Minimalist, aesthetically pleasing design
